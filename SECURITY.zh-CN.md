# 安全政策

[English](SECURITY.md) | [简体中文](SECURITY.zh-CN.md)

请不要在公开 Issue 中披露疑似漏洞。请邮件联系 `tonyhyworld@gmail.com`，并说明受影响版本或 commit、组件、复现步骤、影响与建议缓解方式。不要包含 API Key、客户数据或破坏性载荷。

高优先级报告包括：沙箱逃逸、未经授权的工具或 MCP 访问、凭据泄露、跨用户数据访问、认证或权限绕过、不安全的场景导出、伪造 observation/event/settlement，以及影响查看器的 replay 注入。

目标是在 3 个工作日内确认报告、7 个工作日内完成初步分流；这不是合同服务等级。欢迎避免隐私侵犯、服务中断、数据破坏和未经授权访问的善意安全研究。
