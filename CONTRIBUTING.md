# Contributing to TraceArena

[English](CONTRIBUTING.md) | [简体中文](CONTRIBUTING.zh-CN.md)

Thank you for helping build worlds where agents act, produce evidence, and accept verifiable outcomes.

Before opening a pull request, search existing issues, avoid private data and credentials, and add tests for behavior changes. Keep domain-specific rules in scenario packages rather than the generic runtime. New media or data must include provenance and redistribution rights.

Pull requests should explain the problem, user impact, compatibility implications, tests run, and any security or privacy considerations. AI-assisted contributions are welcome; the human contributor remains responsible for reviewing, testing, and having the right to submit the work.

By contributing, you agree that your contribution is submitted under Apache-2.0. We use DCO sign-off; add `git commit -s` to commits you author.
