# TraceArena Public AI World Replay

Run a bounded, deterministic multi-agent world and inspect its auditable action,
event, and settlement trajectory.

This public Space is deliberately **replay-only**:

- synthetic fixture data;
- no LLM or API key input;
- no brokerage connection or real order execution;
- no MCP, external tools, uploads, or arbitrary code execution;
- no investment advice or promise of future returns.

The complete open-source runtime, documentation, and scenario-pack guides are
available at [tonyhyworld/TraceArena](https://github.com/tonyhyworld/TraceArena).
