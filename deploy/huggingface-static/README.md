# TraceArena AI World Demo

Run a bounded, deterministic multi-agent world in your browser and inspect its
auditable evidence, action, event, settlement, and trajectory digest.

This public Space is deliberately safe and self-contained:

- reviewed synthetic fixture data;
- deterministic browser-local replay;
- no LLM, API key, brokerage, upload, or external tool access;
- no real order execution or investment advice;
- no user data leaves the browser.

The full open-source runtime, backend, operator console, documentation, and
scenario-pack guides are available at
[tonyhyworld/TraceArena](https://github.com/tonyhyworld/TraceArena).
