# 行为准则

[English](CODE_OF_CONDUCT.md) | [简体中文](CODE_OF_CONDUCT.zh-CN.md)

TraceArena 采用 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 作为社区行为准则。

我们期望参与者保持尊重、建设性，并重视隐私与安全。不可接受的行为包括骚扰、歧视、人肉搜索、威胁、故意干扰，以及发布私密凭据或数据。

如需报告行为问题，请联系 `tonyhyworld@gmail.com`。项目会在实际可行范围内私下处理报告。
