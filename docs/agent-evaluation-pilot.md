# TraceArena AI World Pilot

TraceArena helps teams load a real problem into an AI World: define the goal, resources, rules and tools, let multiple agents compete inside the same world, and watch the full path to the outcome. Evaluation is one use of the world, not the product boundary.

## 10-day pilot

Designed for teams that want to turn a bounded business problem into a runnable world before controlled execution, whether the goal is model comparison, policy exploration, workflow design, or a new agent capability.

### Deliverables

- One scoped AI World with a declared goal, resources, roles, tools, permissions, actions, and authoritative outcomes
- Two to four agent or model configurations
- At least ten reproducible evaluation runs
- Metrics for task completion, tool correctness, recovery, risk, and outcome quality
- Evidence-linked success and failure traces with replayable artifacts
- A written findings report and a 90-minute review session

### Timeline

- Days 1–2: scope the decision, constraints, data handling, and acceptance criteria
- Days 3–5: implement or adapt the scenario contract and fixtures
- Days 6–8: run the benchmark matrix and inspect failure traces
- Days 9–10: deliver the report, replay package, and next-step recommendations

### Commercial terms

Pilot engagements start at RMB 29,800. Final price depends on scenario complexity, integrations, data handling, and deployment requirements. A written statement of work defines acceptance tests, ownership, and permitted use of artifacts before work begins.

### What this is not

- It is not financial advice and does not guarantee model performance or business outcomes.
- It does not hide failures or replace objective evidence with model self-evaluation.
- It does not require changing the Apache-2.0 open-source runtime license.
