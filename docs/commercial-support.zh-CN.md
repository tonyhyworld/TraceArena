# 商业支持

TraceArena 是 Apache-2.0 开源项目。核心运行时、场景包契约、回放格式和公开 Demo 保持开放，
商业支持用于帮助团队在开源项目之上完成实施和运营。

## 可提供的服务

- **领域场景设计**：把真实业务转成包含角色、工具、可见性、结构化行动、结算权威、评测指标和回放展示的可验证 AI 世界。
- **世界建模与路径比较**：把业务资源、目标、规则、工具和结果权威装载为场景，在相同约束下比较 Agent 的候选路径，并输出可审计的证据与结算报告。
- **模型与工具集成**：接入模型供应商、MCP 工具、内部 API、沙箱和外部 Agent，同时把凭证留在仓库之外。
- **部署与运营**：适配认证、权限、密钥托管、可观测性、运行归档和企业部署拓扑。

## 合作方式

每个项目先进行需求沟通，再以书面工作说明明确场景契约、集成范围、验收测试、数据处理方式和交付物归属。
开源运行时不是投资建议产品，也不保证模型表现或商业结果。

请发送邮件至 **tonyhyworld@gmail.com**，说明：

- 要建模的领域或工作流；
- 参与的 Agent、模型和工具；
- 必须可验证的证据与结果；
- 期望的部署方式和时间。

如果只需要贡献一个场景包或工具适配器，也可以直接按照开源贡献规则提交，商业支持不是参与生态的前提。
