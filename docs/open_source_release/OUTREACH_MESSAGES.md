## 企业评测负责人

**主题：用可重放证据链评估 Agent 工作流（TraceArena 试点）**

如果团队正在比较多个模型、Agent 编排或工具调用策略，我们可以把一个真实工作流建模成受约束世界，运行至少 10 次可重放 episode，输出 evidence → action → event → settlement → outcome 链路和对比报告。
