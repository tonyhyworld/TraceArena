# TraceArena 社媒发布文案

## 技术开发者版

很多 Agent Demo 只展示“回答得像不像”，却无法回答：它看到了什么？调用了什么工具？
请求的行动是否被世界接受？结果由谁结算？

TraceArena 是一个开源的、可审计的多智能体世界运行时：多个 Agent 在共同规则、有限资源和
真实工具约束下连续行动；世界负责验证行动和结算后果；每一次运行都留下可观看、可解释、可
复现的轨迹。

现在可以直接体验：

- Demo：https://tonyworld888-tracearena-demo.static.hf.space/index.html
- GitHub：https://github.com/tonyhyworld/TraceArena
- 一键安装：`./scripts/install.sh`

如果你要构建一个代码评测、企业运营、治理、科研或策略博弈世界，欢迎贡献场景包。最想看到
的不是一句“很酷”，而是：你会先把哪个世界放进来？

## 企业/研究团队版

如果你的团队正在评估 Agent 的工具使用、长期规划、风险控制或失败恢复，单轮问答和最终分数
通常不够。TraceArena 把观察、证据、工具调用、结构化行动、环境反馈和最终结算放进同一条可
审计链，便于比较、诊断和复用。

如果你有一个具体工作流，希望把它变成可验证的 AI 世界，可邮件联系：
**tonyhyworld@gmail.com**。

## 发布注意

公开 Demo 使用确定性合成数据，不调用 LLM、不连接券商、不接受 API Key，也不构成投资建议。
关于真实模型、内部工具和生产部署，请使用本地完整运行时，并遵守安全与数据治理要求。
