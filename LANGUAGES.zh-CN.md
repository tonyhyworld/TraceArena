# 语言支持

[English](LANGUAGES.md) | [简体中文](LANGUAGES.zh-CN.md)

TraceArena 保持一套运行时和一套场景契约，不维护相互分叉的中英文实现。语言属于呈现层和场景内容层；结果规则、证据、结算和回放标识保持语言无关。

## 当前公开语言

| 范围 | 英文（`en-US`） | 简体中文（`zh-CN`） |
| --- | --- | --- |
| 根 README 与版本信息 | 默认 | 已提供 |
| 贡献与安全说明 | 默认 | 已提供 |
| 无 Key Market Replay CLI 与摘要 | 默认 | 使用 `--locale zh-CN` |
| 合成回放行动理由 | 已提供 | 已提供 |
| 资本市场场景源码包 | 英文说明 | 默认源码语言 |

本版本刻意保留资本市场示例的中文规范场景声明。其 ID、结构化值和结算逻辑均与语言无关。新的公开场景若要宣称支持双语，必须声明支持的 locale，并提供已翻译的用户可见文本。

## Locale 契约

- 使用 BCP 47 语言标签；当前为 `en-US` 与 `zh-CN`。
- ID、动作类型、指标、trace 字段和 fixture schema 必须保持语言无关。
- 标签、prompt、摘要、帮助信息和文档必须本地化。
- 不得翻译证据 ID、哈希、provider ID、证券代码或结算 outcome。

发现翻译缺口时，请提交带 `i18n` 标签的 Issue。
