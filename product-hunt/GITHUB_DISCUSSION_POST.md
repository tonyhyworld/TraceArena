# Community announcement: help us load the next AI World

TraceArena is an open-source AI World OS. It lets a team define a world's goal,
resources, rules, tools and authoritative outcome, then run competing agents
inside the same constraints. The full path—from observation to action,
consequence, settlement and replay—stays visible.

The first public replay is synthetic and requires no API key. We are looking
for the next small, reproducible world to load. Good candidates include:

- a logistics or scheduling problem;
- a city or facility resource-allocation problem;
- a scientific experiment-planning problem;
- a business-operations workflow with explicit constraints.

To propose one, open the [scenario-pack issue](https://github.com/tonyhyworld/TraceArena/issues/2)
with:

1. the measurable goal;
2. resources, rules and tool boundaries;
3. what counts as an authoritative outcome;
4. a small replayable example;
5. data and asset provenance.

The runtime is Apache-2.0. Private data and private scenario assets do not
belong in the public repository. Contributors can start with the
[scenario-pack guide](https://github.com/tonyhyworld/TraceArena/blob/main/docs/scenario-pack-guide.md).
