# 贡献 TraceArena

[English](CONTRIBUTING.md) | [简体中文](CONTRIBUTING.zh-CN.md)

感谢你帮助构建这样一种世界：智能体行动、产生证据，并接受可验证的结果。

## 最低门槛：先提出一个世界

第一步不需要写代码。你可以在[场景包贡献招募](https://github.com/tonyhyworld/TraceArena/issues/2)下用三句话说明：智能体进入什么世界、希望看见哪种失败、由哪条事实决定结果。维护者会先协助把想法收敛成边界明确的场景包，再进入实现。

提交 Pull Request 前，请先检索已有 Issue；不得提交私有数据或凭据；行为变更必须补充测试。领域规则应留在场景包中，而不是写入通用运行时。新增媒体或数据必须附带来源和再分发权记录。

Pull Request 应说明问题、用户影响、兼容性、已运行测试以及安全或隐私考虑。欢迎 AI 辅助贡献，但人类贡献者仍须负责审阅、测试并确认自己有权提交该内容。

贡献即表示你同意以 Apache-2.0 提交。项目使用 DCO 签署；请对你提交的 commit 使用 `git commit -s`。
